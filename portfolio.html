Portfolio




Gordon Shumway: the Memory Game

This memory game requires the user to click on different pictures of ALF without clicking on the same one twice. After each click, all of the images are randomly placed at a new position. Built with: React.JS, JavaScript, HTML, CSS



Front End: Hangman

Traditional hangman game that uses a changing background image as a clue to what the next word is. Built with: JavaScript, HTML, and CSS.



Front End: Giphy API

A website to search for available gifs through the Giphy API. Built with: JavaScript, Giphy API, HTML, CSS.



Back End: Store Manager

A backend mock-up of an online store. It has two components to it, the consumer side and the manager side. The consumer side lets a user pick the merchandise to purchase and quantity, it then only permits the sale if the quantity ordered is in stock. The manager side lets the user add new merchandise, change quantities, and check the quantity of all products. Built with: Node.JS and MySQL.



Database: Burger Menu

A RESTful site that lets the user add a burger to the list/database. They can then update the database by "consuming" the burger changing it's status. Built with: JavaScript, Node.JS, MySQL, Express.JS, HTML, CSS
